使用腾讯云 DNSPod 解析域名

本来想把本网站从 Github Pages 迁移到自己买的云服务器上，但迁移失败并导致原来的解析也无法使用，所以重新在腾讯云上解析，阿里云应该也是同理。

准备工作

  • 腾讯云 DNSPod：管理控制台 - DNSPod
  • Namesilo：Cheap Domain Names & Web Hosting Starting at $0.99! | NameSilo

解析步骤

  1. 打开 DNSPod，登录。左菜单栏找到 我的域名。
  2. 点击 添加域名，输入域名，根据提示操作。
  3. 提示要求换成腾讯云自家的解析服务器。打开 namesilo，登录。
  4. 点击 My account →domain manager。
  5. 选中域名那一栏，点击 Change NameServers。将其改为腾讯云提供的两个服务器。

  1. 如果之前有解析记录，点击蓝色球，进去后全部删掉即可。

  2. 回到 DNSPod，对域名添加记录，只需要两条。

    IPV4 可选四条，其实填一个就行。选了 CNAME 后，一定要在 public 文件夹下创建一个 CNAME 文件，里面只需要写一行自己的域名。

  3. 设置好后等待即可。解析验证需要时间。我设置好后，出去吃了个晚饭，回来就可以了。
