  • 安装 Node.js
  • 安装 Git

安装完成后，Win+R 输入cmd打开，依次输入node -v、npm -v、git --version并回车，出现程序版本号即可。

首先新建一个文件夹来存放 Hexo 的程序文件，如~/Blog/。打开该文件夹，右键，Git Bash Here。
